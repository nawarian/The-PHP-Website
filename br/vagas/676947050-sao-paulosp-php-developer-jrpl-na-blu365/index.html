Descrição da vaga

Aqui na BLU365 você irá construir soluçōes que impactam a vida de milhares de brasileiros, nos ajudando a criar ambientes e soluçōes para que eles voltem a ficar no azul e se mantenham dessa forma nos 365 dias do ano!

Você vai trabalhar com: - Desenvolvimento de aplicaçōes Wordpress (temas e plugins) - Integraçōes com sistemas internos e externos utilizando PHP - Ambiente docker - Consumo e utilizaçāo de serviços AWS

Requisitos

Obrigatórios: - Conhecimentos em PHP7+ - Conhecimento do ambiente PHP: Composer, xdebug, trabalhar com logs, setup VSCode (Ou PHPStorm, se preferir) - Saber utilizar Git / Github - Familiaridade com ambiente Linux e uso de linhas de comando (shell) - Conhecimentos básicos de Vanilla JS, HTML, CSS - Protocolo HTTP e consumo de APIs

Desejáveis: - Conhecimentos de desenvolvimento Wordpress (desenvolvimento de temas e plugins) - Já ter utilizado ReactJS - Se preocupar com performance e organizaçāo do código - Familiaridade com Docker e Nginx
