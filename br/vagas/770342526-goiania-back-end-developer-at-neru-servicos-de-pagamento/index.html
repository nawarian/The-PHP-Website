Descrição da vaga

Estamos procurando profissionais que queiram aprimorar seus conhecimentos em Elixir desenvolvendo tecnologias para meios de pagamento. Você irá trabalhar com os fundadores do projeto desenvolvendo APIs, interfaces Web e integrações com serviços de parceiros.

Local

Goiânia - PJ

Requisitos

Obrigatórios: - Desenvolver, testar e documentar funcionalidades; - Criar interfaces de acordo com as especificações de UX/UI; - Identificar pontos de melhoria no código e na arquitetura para melhorar a experiência dos usuários; - Colaborar com restante da equipe de desenvolvimento para entregar novos produtos; - Colaborar e participar de treinamentos para difusão de boas práticas de desenvolvimento e estar aberto para aprender novas tecnologias.

Diferenciais: - Experiência prévia em Elixir, Ruby ou outra stack de desenvolvimento web moderna; - Banco de dados Postgres SQL; - HTML, CSS e Javascript
