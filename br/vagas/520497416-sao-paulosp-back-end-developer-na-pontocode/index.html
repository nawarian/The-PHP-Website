Descrição da Vaga

Buscamos um profissional para atuar em suporte e desenvolvimento de sistemas PHP, utilizando o framework Laravel.

A pessoa ideal para esta oportunidade deve ser capaz de desenvolver sistemas, APIs e serviços utilizando linguagem e frameworks PHP, será responsável por: * Desenvolver novos sistemas utilizando PHP / MVC * Desenvolver APIs para exposição e consumo de recursos com clientes e parceiros * Manutenção de aplicações legadas desenvolvidas em linguagem PHP * Desenvolver e dar manutenção em robôs de coleta de dados de nossos clientes * Tornar nosso ambiente de trabalho mais divertido e descontraído ;)

Local

São Paulo/SP - Bela Vista - Av. Paulista Próximo ao metrô Consolação

Requisitos

Obrigatórios:

  • Experiência comprovada com desenvolvimento em php 5+
  • Experiência comprovada com laravel 5+
  • Mysql
  • Modelagem em banco de dados relacional
  • Versionamento git
  • Garantir que as entregas correspondem às especificações pré-estabelecida pelo PO
  • Proatividade para resolução de problemas

  • Docker

Diferenciais:
