Nossa empresa

A Jungsoft é uma startup internacional sediada em Florianópolis - SC, com negócios na Alemanha e Estados Unidos. Trabalhamos sempre com tecnologias de ponta, como Elixir, GraphQL, React e React Native.

Possuímos diversos projetos, desde blockchain até big data - ou seja, você vai amar se gostar de desafios e de aprender coisas novas!

Suas responsabilidades aqui serão: - Desenvolver interfaces que trazem uma experiência única e eficiente aos usuários; - Utilizar linguagens como JavaScript e TypeScript para completar suas tarefas; - Realizar a comunicação com APIs GraphQL; - Contribuir com os padrões e melhores práticas; - Fortalecer o ecossistema React local e internacional;
