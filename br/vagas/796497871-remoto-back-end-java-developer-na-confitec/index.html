[Remoto] Back-end Java Developer na Confitec

Descrição da vaga:

Atuação na equipe de projetos e sustentação da Confitec. Você será responsável pelo desenvolvimento e manutenção de sistemas utilizando Java e Spring Boot. Atuará monitorando e analisando a saúde e performance dos serviços.

Confitec:

A Confitec possui fábrica de Software que trabalha com as mais reconhecidas metodologias de desenvolvimento.

Através de processos e procedimentos, nossos produtos são gerados e controlados em todas as fases do processo de construção, levando como ponto principal o atendimento dos requisitos técnicos e funcionais, visando sempre a qualidade e cumprimento dos prazos. Nosso processo de desenvolvimento passa por criteriosas validações de requisitos, baseado em planos de testes bem definidos, bem como um code review dos códigos desenvolvidos por nossos profissionais. Com todo este processo temos obtidos excelentes resultados de qualidade no desenvolvimento.
