PHP 7.5 will not exist!

Leia em Português

I see many people asking in many community spaces when PHP 7.5 will be released.

PHP 7.5 Release Date

According to Nikita's post this February 2020 PHP 7.4 is the last minor version in this major, meaning it will jump from PHP 7.4 to PHP 8.0.

There won't be a PHP 7.5 release.

I think it is time to tweet this to all your PHP dev friends.

PHP 8 is already here, though. Have a look on what's new in this link.
